# 数据库重置工具

## 功能说明

该脚本用于安全地重置整个数据库，包括：

1. **清空数据**：安全删除所有现有数据（正确处理外键约束）
2. **重建表结构**：删除旧表并重新创建所有数据库表
3. **导入默认数据**：插入预设的测试数据

## 使用方法

### 基本用法

```bash
# 进入后端项目目录
cd d:\编程开发\AI\ai_chat\backend

# 使用虚拟环境运行脚本（交互式，需要确认）
.\.venv\Scripts\python.exe reset_database.py
```

### 强制模式（跳过确认）

```bash
.\.venv\Scripts\python.exe reset_database.py --force
```

## 技术特性

### 安全性
- ✅ 执行前会显示警告并要求确认（除非使用 `--force`）
- ✅ 临时禁用外键检查以避免约束冲突
- ✅ 按正确顺序删除数据（从叶子表到主表）
- ✅ 完整的事务处理和错误回滚

### 日志记录
- ✅ 详细的步骤日志输出
- ✅ 每个操作的执行状态
- ✅ 错误信息和堆栈跟踪
- ✅ 日志文件保存位置：`logs/reset_database.log`

### 异步支持
- ✅ 使用异步 SQLAlchemy (aiosqlite)
- ✅ 复用项目的数据库连接池
- ✅ 正确的资源清理和关闭

## 代码结构

```python
reset_database.py
├── disable_foreign_keys()      # 禁用外键检查
├── enable_foreign_keys()       # 启用外键检查
├── clear_all_data()            # 清空所有数据
├── recreate_tables()           # 重建表结构
├── create_default_user()       # 创建管理员用户
├── create_model_provider()     # 创建模型提供商
├── create_models()             # 创建模型列表
├── create_character()          # 创建示例角色
├── create_knowledge_base()     # 创建示例知识库
├── create_global_settings()    # 创建全局设置
├── import_default_data()       # 导入所有默认数据
├── reset_database()            # 主流程控制
└── main()                      # 命令行入口
```

## 注意事项

⚠️ **警告**：此操作是破坏性的，会删除所有现有数据！

- 在执行前务必备份重要数据
- 确保没有其他进程正在使用数据库
- 建议在测试环境中先验证

## 依赖

- Python 3.13+
- SQLAlchemy (异步版本)
- aiosqlite
- 项目虚拟环境已安装所有必要依赖
