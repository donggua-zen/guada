### 设计模式

| 模式 | 应用场景 | 优势 |
|------|----------|------|
| **策略模式** | 多 Provider 路由 | 灵活切换实现 |
| **工厂模式** | 依赖注入 | 解耦创建逻辑 |
| **责任链模式** | 中间件处理 | 可插拔增强 |
