# 按钮位置调整

**调整时间**: 2026-03-27  
**问题**: 按钮挡住了输入框  
**状态**: ✅ 已调整

---

## 🐛 **问题描述**

### 现象
- 回到底部按钮位置偏低（`bottom: 90px`）
- 按钮底部距离输入框太近
- 视觉上显得拥挤，可能遮挡输入框内容

---

## ✅ **调整方案**

### ScrollToBottomButton.vue 修改

**修改前**:
```css
.scroll-to-bottom-btn {
    position: absolute;
    bottom: 90px; /* 输入框上方 */
    /* ...其他样式... */
}
```

**修改后**:
```css
.scroll-to-bottom-btn {
    position: absolute;
    bottom: 120px; /* 输入框上方，留出足够空间 */
    /* ...其他样式... */
}
```

**变化**:
- `bottom: 90px` → `bottom: 120px`
- **向上移动了 30px**
- 留出更多空间给输入框
