## 🎨 色彩一致性测试

### 对比度检查
使用浏览器开发者工具或在线工具检查以下组合的对比度:

- [ ] 主文本 vs 页面背景 ≥ 7:1 (AAA 标准)
- [ ] 次要文本 vs 页面背景 ≥ 4.5:1 (AA 标准)
- [ ] 主色按钮文本 vs 按钮背景 ≥ 4.5:1
- [ ] 用户气泡文本 vs 气泡背景 ≥ 7:1

**推荐工具**:
- WebAIM Contrast Checker: https://webaim.org/resources/contrastchecker/
- Chrome DevTools: Elements > Computed > Contrast ratio

### 色彩层次验证
- [ ] 5 层亮度层级清晰可辨 (L=18%, 20%, 22%, 25-26%, 28%)
- [ ] 相邻层级之间有足够区分度
- [ ] 不会出现"扁平化"或"混乱"的视觉感受
