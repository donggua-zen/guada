## 文档生成规范

- **存放位置**：生成的 `.md` 文档默认应放置于项目根目录的 `docs` 文件夹中。
- **目录结构**：
  - 前端相关文档请存放至 `docs/frontend/` 子目录。
  - 后端相关文档请存放至 `docs/backend/` 子目录。
- **例外情况**：若指令中特别指定了其他路径，则以指定路径为准。

## 注释规范
- 禁止在代码注释中使用 emoji 表情符号。
- 注释应使用清晰、简洁的中文或英文描述，保持专业性和可读性。

## 数据库迁移规范
- 必须使用虚拟环境
- 必须使用 `alembic revision -m "xxxx"` 命令生成迁移脚本。
- 禁止手动创建迁移文件，以确保迁移历史的一致性和完整性。
