# 知识库树形文件浏览器使用指南

## 功能概述

知识库现在支持**两种视图模式**:
1. **列表视图** - 传统的扁平化文件列表
2. **树形视图** - 类似Windows资源管理器的文件夹树形结构

---

## 使用方法

### 切换视图模式

在知识库页面的"文件列表"Tab右上角,有两个按钮:

```
[📋 列表] [🌳 树形]
```

- 点击 **📋 列表** - 切换到传统列表视图
- 点击 **🌳 树形** - 切换到树形文件夹视图

### 树形视图操作

#### 展开/折叠文件夹
- **点击文件夹名称** - 展开或折叠该文件夹
- 展开的文件夹会显示蓝色背景
- 箭头图标会旋转90度表示展开状态

#### 查看文件
- **点击文件** - 如果文件处理完成,会打开文件详情
- 文件右侧显示大小和状态标签

#### 空文件夹
- 展开空文件夹会显示 "(空文件夹)" 提示

## 技术实现

### 核心组件

1. **KBFileTree.vue** - 树形视图容器
   - 将扁平文件列表转换为树形结构
   - 递归渲染所有节点
   - 自动排序(文件夹优先)

2. **TreeNode.vue** - 单个树节点
   - 支持展开/折叠
   - 递归渲染子节点
   - 显示文件/文件夹信息

### 数据结构转换

```typescript
// 输入: 扁平列表
[
  { id: '1', displayName: 'A', isDirectory: true, parentFolderId: null },
  { id: '2', displayName: 'B', isDirectory: true, parentFolderId: '1' },
  { id: '3', displayName: 'file.txt', isDirectory: false, parentFolderId: '2' }
]

// 输出: 树形结构
{
  id: '1',
  displayName: 'A',
  children: [
    {
      id: '2',
      displayName: 'B',
      children: [
        { id: '3', displayName: 'file.txt', children: [] }
      ]
    }
  ]
}
```

### 排序规则

1. **文件夹优先** - 文件夹始终排在文件前面
2. **按名称排序** - 同级节点按displayName字母顺序排序
3. **递归应用** - 所有层级都应用相同排序规则

## 常见问题

### Q1: 为什么有些文件没有显示在文件夹下?

**A:** 检查以下几点:
1. 文件的 `parentFolderId` 是否正确指向父文件夹
2. 父文件夹是否存在于当前列表中
3. 刷新页面重新加载数据

### Q2: 文件夹展开后如何折叠?

**A:** 再次点击文件夹名称即可折叠。

### Q3: 能否同时展开多个文件夹?

**A:** 可以!每个文件夹的展开/折叠状态是独立的。

### Q4: 树形视图支持分页吗?

**A:** 当前版本不支持。树形视图会显示所有文件。如果文件数量很大(>500),建议使用列表视图。

### Q5: 删除文件夹会发生什么?

**A:** 后端会级联删除该文件夹下的所有子文件夹和文件。

## 性能优化建议

### 大量文件场景

如果知识库包含超过500个文件:

1. **使用列表视图** - 性能更好
2. **启用搜索** - 快速定位文件
3. **分批上传** - 避免一次性上传过多文件

### 深层嵌套场景

如果文件夹嵌套超过5层:

1. **考虑重构目录结构** - 减少嵌套深度
2. **使用搜索功能** - 直接查找目标文件
3. **记住常用路径** - 减少展开操作

## 开发者说明

### 添加新字段到树节点

如果需要显示更多信息,修改 `TreeNode.vue`:

```vue
<!-- 在节点内容区域添加 -->
<span v-if="node.customField" class="text-xs text-gray-500">
    {{ node.customField }}
</span>
```

确保数据源包含该字段:
```typescript
// KnowledgeBasePage.vue 中的数据映射
files.value = dbFiles.map(file => ({
    ...file,
    customField: file.customField, // ← 添加新字段
}))
```

### 自定义排序规则

修改 `KBFileTree.vue` 中的 `sortNodes` 函数:

```typescript
const sortNodes = (nodes: any[]) => {
    nodes.sort((a, b) => {
        // 自定义排序逻辑
        if (a.isDirectory && !b.isDirectory) return -1
        if (!a.isDirectory && b.isDirectory) return 1
        
        // 例如: 按文件大小排序
        // return b.fileSize - a.fileSize
        
        // 默认: 按名称排序
        return a.displayName.localeCompare(b.displayName)
    })
    
    nodes.forEach(node => {
        if (node.children.length > 0) {
            sortNodes(node.children)
        }
    })
}
```
